# Pathway-GEM-scripts
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20522432.svg)](https://doi.org/10.5281/zenodo.20522432)

This repository contains pathway-specific GEM Colab notebooks for reduced symbiont metabolic modeling.  
Scripts included:
- Arginine pathway specific model
- Threonine pathway specific model
- Tryptophan pathway specific model
- Lysine pathway specific model
- Leucine pathway specific model
- Phenylalanine pathway specific model
- Each notebook was created in Google Colab and saved here for reproducibility and DOI citation via Zenodo.
